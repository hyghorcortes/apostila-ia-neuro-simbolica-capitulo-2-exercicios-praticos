# Exercício Prático 5 — Auditoria Simbólica sob Concept Drift

## Objetivo
Mostrar como uma regra simbólica pode deixar de funcionar adequadamente quando o ambiente muda.

## Conceitos trabalhados
- auditoria de regras
- aderência histórica versus contexto novo
- necessidade de revisão do conhecimento explícito

## Arquivo principal
- `exercicio_pratico_5_auditoria_concept_drift.py`

## Como executar
```bash
python exercicio_pratico_5_auditoria_concept_drift.py
```

## Resultado esperado
O programa compara o desempenho da regra em dados antigos e novos, destacando a necessidade de atualização.
