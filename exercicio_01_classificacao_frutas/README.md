# Exercício Prático 1 — Classificação Simbólica de Frutas

## Objetivo
Construir uma pequena base de conhecimento com fatos e regras para classificar objetos como frutas.

## Conceitos trabalhados
- representação simbólica
- fatos binários
- regras explícitas
- inferência por casamento de condições

## Arquivo principal
- `exercicio_pratico_1_base_conhecimento_frutas.py`

## Como executar
```bash
python exercicio_pratico_1_base_conhecimento_frutas.py
```

## Resultado esperado
O programa avalia regras simbólicas e informa a classificação final de cada objeto.
