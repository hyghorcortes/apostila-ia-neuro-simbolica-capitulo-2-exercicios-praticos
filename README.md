# Capítulo 2 — Exercícios Práticos de IA Simbólica

Este repositório reúne **5 exercícios práticos** prontos para importação direta no GitHub.

## Estrutura

- `exercicio_01_classificacao_frutas/`: base de conhecimento simbólica para classificação de frutas
- `exercicio_02_recomendador_filmes/`: recomendação simbólica de filmes por regras
- `exercicio_03_agendador_restricoes/`: satisfação de restrições em agenda
- `exercicio_04_sistema_especialista_manutencao/`: sistema especialista com explicação da decisão
- `exercicio_05_auditoria_concept_drift/`: auditoria simbólica de regras sob mudança de contexto

## Como executar

Todos os exemplos usam apenas a **biblioteca padrão do Python**.

Exemplo:

```bash
python exercicio_01_classificacao_frutas/exercicio_pratico_1_base_conhecimento_frutas.py
```

## Objetivo didático

Os exercícios foram organizados para mostrar, de forma prática, conceitos centrais da IA simbólica, como:

- representação explícita do conhecimento
- regras lógicas
- inferência baseada em fatos
- explicabilidade
- revisão de regras em ambientes mutáveis
