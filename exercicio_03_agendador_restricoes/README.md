# Exercício Prático 3 — Agendador com Restrições

## Objetivo
Resolver um problema simples de agendamento usando restrições simbólicas.

## Conceitos trabalhados
- satisfação de restrições
- verificação lógica de condições
- justificativa de inviabilidade

## Arquivo principal
- `exercicio_pratico_3_agendador_restricoes.py`

## Como executar
```bash
python exercicio_pratico_3_agendador_restricoes.py
```

## Resultado esperado
O programa identifica horários válidos e explica por que determinados horários foram rejeitados.
