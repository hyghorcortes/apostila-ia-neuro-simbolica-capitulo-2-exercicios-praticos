# Exercício Prático 2 — Recomendador Simbólico de Filmes

## Objetivo
Aplicar regras lógicas para recomendar ou não recomendar filmes com base em atributos previamente definidos.

## Conceitos trabalhados
- regras booleanas
- tomada de decisão simbólica
- explicação da decisão

## Arquivo principal
- `exercicio_pratico_2_recomendador_filmes_simbolico.py`

## Como executar
```bash
python exercicio_pratico_2_recomendador_filmes_simbolico.py
```

## Resultado esperado
O programa informa quais filmes são recomendados e por que alguns são recusados.
