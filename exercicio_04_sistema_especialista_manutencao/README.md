# Exercício Prático 4 — Sistema Especialista de Manutenção

## Objetivo
Simular um sistema especialista simples capaz de inferir recomendações de manutenção a partir de fatos observados.

## Conceitos trabalhados
- regras de produção
- trilha de inferência
- explicabilidade

## Arquivo principal
- `exercicio_pratico_4_sistema_especialista_manutencao.py`

## Como executar
```bash
python exercicio_pratico_4_sistema_especialista_manutencao.py
```

## Resultado esperado
O sistema apresenta a conclusão final e a sequência de regras que sustentam a decisão.
